## 借鉴Spring的思想
- Spring的不足
    - 手动Wire:配置太过庞大
    - 自动Wire:复杂项目无法实现

## 不同于Spring
- 取消xml
- 取消bean的概念
- 使用Constructor来注入
- 泛型支持
- 一个专注于Dependency Injection的框架
